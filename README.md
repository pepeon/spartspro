# SPartsPro

**SPartsPro** is a static website hosted on **GitHub Pages**. It serves as a product catalog and company information page for a supplier of elevator cables, belts, and related spare parts.

The site provides information about available products, the company, and contact details in a simple and accessible format.

---

## 🌐 Live Demo
The project is deployed using **GitHub Pages**.

https://pepeon.github.io/spartspro/

---

## 📌 Features

- **Responsive (adaptive) design** — optimized for desktops, tablets, and mobile devices.
- **Hamburger navigation menu** for convenient navigation on small screens.
- **Product catalog** displaying elevator cables and belts.
- **Company information** section.
- **Contact details** for customers and partners.
- Lightweight and fast loading.
